Module: skyring/lib/nats

Small wrapper around nats for quickly connecting / disconnecting

Since:
  • 1.0.0
Author:
  • Eric Satterwhite
Source:
  • lib/nats.js

Example

const nats= require('skyring/lib/nats')
const client = nats.createClient()
client.publish('foobar', JSON.stringify({'foo':'bar'}), () => {
  nats.quit()
})
